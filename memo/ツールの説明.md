# ツールの説明

## 概要

音声ファイル（WAV）とテキストを入力として、
モーラ単位の音声特徴量テーブルをJSONで出力するツール。

---

## パイプライン（現在の流れ）

```
入力: input/001.wav + テキスト
  │
  ├─ whisper          音声 → テキスト
  ├─ text_normalize   テキストを正規化
  ├─ text_replace     whisperの誤認識を辞書で置換
  ├─ 004 text_clean   neologdnで整形
  │
  │  ※ 005(text_to_kana) / 006(text_phoneme) / 007(text_mfa_phoneme) は現在カット
  │
  ├─ 008 run_mfa_alignment()       MFAで音声と音素を整合（タイムスタンプ付与）
  ├─ 009 run_phoneme_to_mora()     音素列 → モーラ列に変換
  ├─ 010 run_mora_to_japanese()    モーラを日本語表記に変換
  ├─ 011 run_librosa_analysis()    モーラごとに音声特徴量を抽出
  └─ 012 run_audio_table_pipeline() TextGrid → 単語×モーラ整合 → JSON出力
```

---

## 各ステップの詳細

| ステップ | モジュール | 内容 |
|---|---|---|
| whisper | `src/whisper/` | 音声をテキストに変換 |
| normalize | `src/text_normalize/` | テキストをUnicode正規化 |
| replace | `src/text_replace/` | dict/replace_dict.txt の辞書で誤認識を置換 |
| 004 | `src/text_clean/` | neologdnで整形 |
| 008 | `src/text_mfa_timestamp/` | Montreal Forced Aligner で音素にタイムスタンプを付与 |
| 009 | `src/text_mora/` | 音素列をモーラ単位にまとめる |
| 010 | `src/text_mora_jp/` | モーラをひらがな等の日本語表記に変換 |
| 011 | `src/audio_analysis/` | librosaでモーラごとの特徴量を抽出（duration, rms, f0, spectral_centroid, zcr） |
| 012 | `src/audio_table/` | TextGridを単語分割 → モーラとマージ → JSONに変換 |

---

## 出力

- `confirm/` 以下に各ステップの中間ファイルが出力される
- 最終出力は `out_json/` にJSON形式で保存される
