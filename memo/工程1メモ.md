Whisperで音声ファイルをテキストにする

pip install openai-whisper で whisperを入れる
winget install ffmpeg で ffmpeg を入れる

実行コマンド
whisper input/001.wav --language Japanese --model small
smallは結構長いかも1分くらい
実行すると、
001.json
001.srt
001.tsv
001.txt
001.vtt
ができる
精度は良い
